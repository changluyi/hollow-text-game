# 虚渊文字 | Hollow Text

> 在黑暗中前行，以魂为灯

一款受《黑暗之魂》系列启发的文字 Roguelike 游戏。

## 🎮 在线游玩

直接访问 GitHub Pages 即可游玩：
```
https://<你的用户名>.github.io/<仓库名>/
```

## 🖥️ 本地运行

### 终端版
```bash
cd hollow_text
pip install rich
python3 main.py
```

### 网页版
直接用浏览器打开 `hollow_text/web/index.html`

## 🎯 游戏特色

- **魂风格回合制战斗** - 每次攻击都是决策，贪刀会死
- **精力管理系统** - 攻击、防御、闪避都需要消耗精力
- **敌人行为模式** - 观察敌人的蓄力和后摇，把握攻击时机
- **死亡惩罚** - 死亡掉落魂，需返回死亡地点取回
- **篝火系统** - 休息恢复但敌人会复活

## 🎮 操作说明

### 探索模式
- 点击方向按钮移动
- 在篝火处可以休息或升级

### 战斗模式
| 行动 | 精力消耗 | 效果 |
|------|---------|------|
| 攻击 | 6 | 普通攻击 |
| 重击 | 10 | 高伤害但会硬直 |
| 防御 | 3 | 减少70%伤害 |
| 闪避 | 5 | 有几率完全躲避 |
| 逃跑 | 0 | 尝试逃离战斗 |

### 战斗技巧
- 敌人**蓄力时**是攻击的最佳时机（额外伤害）
- 敌人**喘息时**也是脆弱状态
- 重击威力大但会让你暴露破绽
- 精力管理是生存的关键
